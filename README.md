# Is it just allergic rhinitis? Predictive modeling to identify biomarkers associated with asthma comorbidity using both clinical and airway microbiome data

Our study developed machine learning models to predict whether
patients with allergic rhinitis (AR) also had asthma comorbidity
(ARAS), given patients’ clinical measures and airway microbiome
abundances. Logistic regression, stepwise selection, LASSO, and
random forest yielded models with between 70-78% accuracy; the
highest was achieved by a random forest which used a subset of
predictors from stepwise selection. Interpreting the resulting models
yielded a subset of potential key predictors that included both clinical
measures and microbiomes.
